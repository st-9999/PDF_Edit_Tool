## P3. 編集機能 (v1)

- [ ] **並び替え**: `dnd-kit` でサムネD&D → `reorder` 操作発行

- [ ] 右クリック `ContextMenu`（回転/削除/分割/抽出）
- [ ] 文脈依存のツールバー活性制御（選択時のみ有効）
- [ ] **テスト（pdf-libで実出力を検証）**: 並び替え後のページ順 / 回転角がページ辞書に反映 / 削除後の総ページ数 / 抽出PDFのページ内容一致 / 分割の境界 / 結合の総ページ数と順序
